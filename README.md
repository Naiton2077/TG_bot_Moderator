Самый обычный бот модератор для чатов.

В его компитенциях {
  (Определяет админа)
  Приметствовать новых участников с кратким шаблоном правил данной беседы
  Прощаться с усшедшими и нарушителями
  Выдавать БАН
  Мутить с возможностью выбрать время мута
  Ночной режим: с 22:00 до 8:00 по МСК сообщения приходят тихо. автоматически выключаются уведомления
  Выдавать предупреждения: (каждый 3 пред = 1 выговор) (каждый 3 выговор = автоБАН) Так же есть возможность снимать предупреждения и выговоры
  АнтиМАТ: автоматически удаляет сообщение содержащие нецензурное слово. либо редактирует сообщения так чтобы не оставалось мата.
}

Проблемные моменты {
  Муты работают не коректно или вообще не работают
  АнтиМАТ: пропускает сообщение содержащие 2 матерных слова подряд. При этом если идёт ещё хорошие слово то он это сообщение редактирует. А если идёт просто 1 мат то он его удаляет. по логике сообщение с 2 матами должно быть удаленно...
}
